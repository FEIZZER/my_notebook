### html里的特殊字符

```
	空格符		：	&nbsp			// 默认页面中文本的空格会被忽略掉
	<			：	&lt				// 小于符号
	>			：	&gt				// 大于符号
	&			：	&amp			
	￥			：	&yen			//人民币符号
	©			：	&copy			// 版权符号
	®			：	&reg			// 注册商标符号
	°			：	&deg			// 度的符号
	±			：	&plusmn			// 加减的符号
	x			：	&times			// 乘号
	÷			：	&divide			// 除号
	²			：	&sup2			// 平方的右上标
	³			：	&sup3			// 立方的右上标
```
