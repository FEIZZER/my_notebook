## 效率问题

1.  用  ++i  代替  i++
2.  使用string类的加法时，用str1+=str2  代替  str1=str1+str2
3.  if语句中判定条件写完整

## 输出小数位数

```
#include <iostream>
#include <iomanip>
using namespace std;
int main()
{
    const double PI=3.1415926;
    cout<<setprecision(7)<<fixed<<PI;//输出7位小数
    cout<<setprecision(7)<<PI;//输出保留7位有效数
}
```
